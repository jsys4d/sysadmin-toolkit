# SysAdmin Toolkit

Toolkit personal de administración de sistemas para entornos Linux.

## Características

- Aliases organizados (network, system)
- Script de instalación automática
- Salida formateada y con colores
- Portable entre sistemas

## Instalación

```bash
git clone https://github.com/TU_USUARIO/sysadmin-toolkit.git
cd sysadmin-toolkit
chmod +x install.sh
./install.sh
